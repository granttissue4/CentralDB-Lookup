# Changelog

All notable changes to this fork are documented here. Bump `version` in `manifest.json` for each release you share.

## [3.0.1] - 2026-04-02

- Added repository docs: README, LICENSE (MIT), this changelog.
- Generated placeholder toolbar icons (`icons/` 16 / 48 / 128) so the manifest resolves.
- Removed unused `content.js` (not listed in `manifest.json`; functionality covered by popup + `content-centraldb.js` / `bridge.js`).
- Documented optional `debug.html` tooling, permissions, Firefox load steps, smoke tests, and optional upstream Git workflow.
